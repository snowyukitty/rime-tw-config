# 台灣正體字形對照表（學習用）

> 用途：打拼音、要台灣標準字輸出時，常見的「選錯字 / 舊字形」對照。
> 分三類，性質不同，解法也不同。本檔對應本專案 `rime-config/opencc/tw_fix.txt`
> 與 `patch-luna-dict.ps1` 的實際處理。

字形標準以**中華民國教育部《國字標準字體》**為準（俗稱「台灣正體 / 正字」）。

---

## 一、異體字 / 舊字形（同音同義、只是字形不同）

這類左右兩字**意思完全一樣**，只是寫法新舊/地區差異。台灣正體只用右欄那個。
luna_pinyin 詞庫沿用了左欄舊字形，而 OpenCC 的 `t2tw` 只收了一部分，**漏掉了「爲」等**
——這就是本專案最初「無能爲力」問題的根源。以下 15 組是本專案已就地修正的：

| 舊字形 / 異體 | 台灣正字 | 例詞 | 備註 |
|:---:|:---:|---|---|
| 爲 | **為** | 因為、行為、認為 | ★最初的問題字；t2tw 不會轉 |
| 着 | **著** | 著急、睡著、顯著 | 「着」為陸/港字形 |
| 裏 | **裡** | 這裡、心裡、裡面 | 「裏」台灣也通，「裡」較常用 |
| 麪 | **麵** | 麵包、泡麵、麵條 | |
| 羣 | **群** | 群組、人群、族群 | |
| 衆 | **眾** | 眾多、觀眾、群眾 | |
| 牀 | **床** | 起床、床鋪、病床 | |
| 啓 | **啟** | 開啟、啟動、啟發 | |
| 麽 | **麼** | 什麼、怎麼、那麼 | 注意：簡體是「么」 |
| 綫 | **線** | 路線、電線、底線 | |
| 峯 | **峰** | 山峰、高峰、巔峰 | |
| 鷄 | **雞** | 雞蛋、公雞、雞肉 | |
| 鬬 / 鬭 | **鬥** | 戰鬥、奮鬥、鬥爭 | |
| 飈 | **飆** | 飆車、狂飆、飆升 | |

> 這 14 列（鬬/鬭 併為一條）即 15 組替換規則。挑選原則：**純字形、無語義分裂、
> 台灣正體為單一標準字**。刻意排除了 `卻`（本身就是正字）、`污/汙`、`慾/欲`、
> `喫/吃`、`棱/稜`、`廼` 等「有語義差或實務通用」者，以免誤傷。

## 二、一簡多繁：常見選錯（最容易出錯的一類）

這類是**一個簡體字對應多個不同意思的繁體字**。打拼音時輸入法多半靠詞組判斷，
但單字或冷詞容易**選錯**。左為簡體，右為「依語義要選的正字」：

| 簡體 | 台灣正體（依語義分流） | 正確示例 | 常見錯誤 |
|:---:|---|---|---|
| 后 | **後**(之後/後面) ・ **后**(皇后/太后) | 以後、後面 ／ 皇后 | 以后 ✗ |
| 里 | **裡/裏**(裡面) ・ **里**(公里/鄰里/故里) | 這裡、心裡 ／ 公里 | 這里 ✗ |
| 面 | **麵**(麵食) ・ **面**(臉/表面/方面) | 麵包、泡麵 ／ 表面 | 泡面 ✗ |
| 干 | **乾**(乾燥) ・ **幹**(幹部/才幹/幹活) ・ **干**(干擾/天干) | 乾淨 ／ 幹勁 ／ 干涉 | 干净 ✗ |
| 发 | **髮**(頭髮) ・ **發**(發生/出發) | 頭髮、理髮 ／ 發現 | 头发→頭發 ✗ |
| 松 | **鬆**(輕鬆/蓬鬆) ・ **松**(松樹/松鼠) | 輕鬆、放鬆 ／ 松樹 | 輕松 ✗ |
| 系 | **係**(關係) ・ **繫**(聯繫/繫鞋帶) ・ **系**(系統/科系) | 關係 ／ 聯繫 ／ 系統 | 關系 ✗ |
| 钟 | **鐘**(時鐘/分鐘) ・ **鍾**(姓鍾/鍾愛) | 五分鐘、鬧鐘 ／ 鍾情 | 五分鍾 ✗ |
| 历 | **歷**(經歷/歷史) ・ **曆**(日曆/農曆) | 歷史、經歷 ／ 月曆 | 月歷 ✗ |
| 复 | **複**(複雜/重複) ・ **復**(恢復/復習) ・ **覆**(覆蓋/答覆) | 複雜 ／ 恢復 ／ 答覆 | 恢複 ✗ |
| 制 | **製**(製造/製作) ・ **制**(制度/控制) | 製造、製作 ／ 制度 | 制造 ✗ |
| 只 | **隻**(一隻/船隻) ・ **只**(只是/只有) | 一隻貓 ／ 只有 | 一只貓 ✗ |
| 台 | **臺/台**(臺灣/舞台) ・ **颱**(颱風) ・ **檯**(檯燈/櫃檯) | 臺灣、舞台 ／ 颱風 ／ 櫃檯 | 台风→台風 ✗ |
| 表 | **錶**(手錶/鐘錶) ・ **表**(表格/外表) | 手錶 ／ 表格、外表 | 手表 ✗ |
| 周 | **週**(週末/一週/週期) ・ **周**(周圍/周到/姓周) | 週末、一週 ／ 周圍 | 周末（兩通，正式用週）△ |
| 板 | **闆**(老闆) ・ **板**(木板/地板) | 老闆 ／ 木板 | 老板 ✗ |
| 注 | **註**(註冊/註解/備註) ・ **注**(注意/注射) | 註冊、備註 ／ 注意 | 注册→注冊 ✗ |
| 志 | **誌**(雜誌/日誌) ・ **志**(志願/同志) | 雜誌 ／ 志願 | 雜志 ✗ |
| 谷 | **穀**(穀物/五穀) ・ **谷**(山谷/峽谷) | 稻穀、五穀 ／ 山谷 | 五谷 ✗ |
| 须 | **鬚**(鬍鬚/觸鬚) ・ **須**(必須/須知) | 鬍鬚 ／ 必須 | 必鬚 ✗ |
| 范 | **範**(範圍/模範) ・ **范**(姓范) | 範圍、示範 ／ 姓范 | 范圍 ✗ |
| 余 | **餘**(剩餘/多餘) ・ **余**(姓余/文言「我」) | 剩餘、業餘 ／ 姓余 | 剩余 ✗ |
| 丑 | **醜**(醜陋/美醜) ・ **丑**(丑角/子丑) | 醜陋 ／ 小丑、丑時 | 醜角 ✗ |

> △「週/周」兩字台灣皆通，但「星期」義正式寫法用「週」。
> 訣竅：**這類靠「詞」不靠「字」**——輸入完整詞組（如 `yihou`→以後、`mianbao`→麵包），
> 輸入法幾乎都會給對；單獨打一個字再湊，才容易選到另一義的字。

## 三、台灣正體 vs 其他繁體（補充，未必出現在詞庫）

同屬「繁體」，但台灣／香港／大陸繁的字形取向不同。第一類其實就是這種差異的一部分。
常見傾向：

| 概念 | 台灣正體 | 其他繁體常見 | 說明 |
|---|:---:|:---:|---|
| wéi（做為） | 為 | 爲 | 見第一類 |
| zhe（…著） | 著 | 着 | 香港多用「着」 |
| zhòng（眾） | 眾 | 衆 | |
| qún（群） | 群 | 羣 | 香港多用「羣」 |
| 部首「丷/八」 | 為、眾 | 爲、衆 | 上半寫法差異 |

> 工具層面：OpenCC `t2tw.json` 用 `TWVariants` 處理這類，但**收錄不全**
> （「爲」就漏了），所以本專案改採「直接修詞庫 + tw_fix 補一道」雙保險。

---

## 四、本專案怎麼落地（對應檔案）

1. **基礎詞庫直改**（主修法，跨所有 schema 生效）：
   `rime-config/patch-luna-dict.ps1` 把上面第一類 15 組就地替換進
   小狼毫共享詞庫 `luna_pinyin.dict.yaml`，重新部署後編譯表即不含舊字形。
2. **OpenCC 輸出層補正**（防護網，僅 `luna_pinyin_tw`）：
   `rime-config/opencc/tw_fix.txt` + `tw_fix.json`，在 `t2tw` 之後再轉一次。
3. **自學詞庫**：用同一張對照表就地替換（保留詞頻、非刪除）。

> 第二、三類「一簡多繁」屬語義分流，**不適合**用無條件字元替換（會把
> 「皇后」改成「皇後」），只能靠**詞庫詞組**正確收錄；故本專案不對它們做全域替換，
> 僅列於此供學習與選字參考。
