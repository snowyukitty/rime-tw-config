# Rime（小狼毫）vs 微軟拼音 詳細比較

> 情境：Windows 11、全拼、要台灣標準字、重視穩定/省資源/可控。
> 標示：✅優 / 🟡普通 / ❌弱。結論為「此情境下」的判斷，非絕對。

---

## 一、輸入體驗

| 項目 | Rime（朙月拼音 全拼） | 微軟拼音 |
|---|---|---|
| 拼音方案 | ✅ 全拼/雙拼/模糊音皆可，方案自由換 | ✅ 全拼/雙拼/模糊音內建 |
| 整句智能組句 | 🟡 中等（內建詞庫較保守，可換 rime-ice 變強） | ✅ 強（微軟語言模型成熟） |
| 雲端候選預測 | ❌ 無（設計上純本地） | ✅ 有（連網時更聰明，但有隱私代價） |
| 詞庫規模 | 🟡 內建中等；可自由擴充/換大詞庫 | ✅ 內建大、含流行詞 |
| 自學習/個人化 | ✅ 本地 user_dict 自學，可清可控 | ✅ 自學，但雲端會反覆覆寫 |
| 英文混輸 | ✅ Shift 上屏；可裝英文方案 | ✅ 內建中英混輸 |
| Emoji/特殊符號 | 🟡 需配置（符號表/emoji 濾鏡） | ✅ 內建 emoji/U 模式/V 模式 |
| 造詞/自訂詞 | ✅ 純文字字典，批次管理、可版控 | 🟡 有自訂短語，但管理陽春 |

**小結：** 開箱即用的「聰明度」微軟略勝（靠雲端＋大模型）；
但 Rime 的個人化是「你完全掌控」，且可升級 rime-ice 補足整句能力。

---

## 二、繁體 / 字形（你的核心痛點）

| 項目 | Rime | 微軟拼音 |
|---|---|---|
| 預設輸出 | ✅ 繁體（朙月詞庫本身繁體） | ❌ 簡體核心，繁體靠輸出轉換 |
| 台灣標準字（為/眾/裡/麼） | ✅ 掛 `t2tw` 永久精準鎖定 | ❌ 字形控制差，易出「爲/衆/裏」 |
| 「爲→為」這種單一錯誤 case | ✅ 一行設定/一條字典永久修正 | ❌ 黑箱模型，改不到、會被洗回 |
| 簡繁一鍵切換 | ✅ 狀態列開關 | 🟡 需在設定切換輸出模式 |

**小結：** 這一類 Rime **完勝**——正是你想做卻在微軟做不到的事。

---

## 三、穩定性 / 資源

| 項目 | Rime（0.17.4 穩定版） | 微軟拼音 |
|---|---|---|
| 記憶體/CPU 占用 | ✅ 輕量 | ✅ 系統內建、極輕 |
| 崩潰史 | 🟡 0.16.x 曾大崩潰（避開即可）；偶發 WeaselServer crash | ✅ 極穩定，系統級維護 |
| 系統管理員視窗 | 🟡 少數情況切不到中文 | ✅ 一律可用 |
| UWP / 部分全螢幕遊戲 | 🟡 偶有異常（如 Fortnite） | ✅ 相容性最佳 |
| 開機/登入即用 | ✅ 安裝後常駐 | ✅ 原生 |

**小結：** 純「絕不出事」微軟略勝；但 Rime 用對版本（0.17.4）後，
日常場景穩定度足夠，差距只在 admin/遊戲等邊角。

## 四、客製化 / 控制權

| 項目 | Rime | 微軟拼音 |
|---|---|---|
| 設定彈性 | ✅ 幾乎全可調（鍵位/候選/濾鏡/方案） | ❌ 選項很少 |
| 精準改錯誤候選 | ✅ 可刪可加可加權 | ❌ 幾乎不行 |
| 外觀（字型/配色/橫直排） | ✅ 完全自訂 | 🟡 只有少數內建主題 |
| 快捷鍵自訂 | ✅ 可重綁 | 🟡 有限 |
| 設定可版控/同步 | ✅ 純文字檔，git 一鍵還原 | ❌ 綁帳號/系統 |

**小結：** 控制權 Rime **完勝**，代價是要花時間配置（已幫你配好）。

---

## 五、隱私 / 安全

| 項目 | Rime | 微軟拼音 |
|---|---|---|
| 輸入資料上雲 | ✅ 純本地，不外傳 | ❌ 雲候選/遙測會傳輸入資料 |
| 遙測 | ✅ 無 | ❌ 有（可關但預設多開） |
| 廣告/推廣 | ✅ 無 | ✅ 無（比搜狗/百度乾淨） |
| 開源可審 | ✅ 完全開源 | ❌ 閉源 |

**小結：** 隱私 Rime **完勝**。

---

## 六、維護 / 生態 / 上手

| 項目 | Rime | 微軟拼音 |
|---|---|---|
| 更新頻率 | 🟡 慢（穩定版停在 2025/6，但夠穩） | ✅ 隨 Windows Update |
| 官方客服 | ❌ 社群為主 | ✅ 微軟支援 |
| 跨平台 | ✅ Win/Mac/Linux/Android/iOS 同生態 | ❌ 僅 Windows |
| 安裝 | 🟡 要自己裝＋部署 | ✅ 免安裝 |
| 學習曲線 | ❌ 陡（設定門檻高） | ✅ 零學習 |

**小結：** 省心/免維護微軟勝；自由/跨平台 Rime 勝。

---

## 總評（你的情境）

| 維度 | 贏家 |
|---|---|
| 台灣字形 / 精準修正 | **Rime（決定性）** |
| 隱私 / 無雲端 / 開源 | **Rime** |
| 客製化 / 控制權 / 版控 | **Rime** |
| 開箱聰明度（雲端整句） | 微軟 |
| 絕對穩定 / admin / 遊戲相容 | 微軟（小幅） |
| 免安裝 / 零學習 | 微軟 |

**一句話：** 你最在意的「台灣字形＋穩定省資源＋可控」，Rime 在「字形/控制/隱私/省資源」全面領先，只在「邊角相容性」與「開箱聰明度」略遜——而後者可用 rime-ice 補足、前者一個月遇不到幾次。**對你而言，Rime 完全可取代微軟拼音，且是升級而非降級。**

**唯一保留情境：** 偶發的系統管理員視窗 / 特定全螢幕遊戲 → 臨時切回系統內建即可。
